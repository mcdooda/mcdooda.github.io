Personal Projects

Flat Engine

Flat Engine was initially designed for Crispy Guacamole, a 2D isometric Real Time Strategy game but its core systems make it generic enough to be used for any type of game.

Both Lua and Visual Scripting can be used for implementing Component-Based Entities and the User Interface.

It supports input handling from the mouse, keyboard and multiple gamepads.

OpenGL is used for rendering along with GLSL shaders.

The engine also offers its own custom performance profiler and Lua memory analyzer.

Crispy Guacamole

Crispy Guacamole is a 2D isometric Real Time Strategy game.

It is a good proof of what Flat Engine can do: Pathfinding on a dynamic terrain, a generic State Machine Graph system for both Animation and Behavior.

The Level Editor along with the Entity Editor feature Lua and visual scripts hot reloading for fast iteration. They have proved to be powerful enough to be used for other genres such as a Tower Defense, a Clicker game and a Shooter / Action game.
